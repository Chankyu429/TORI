# BMP

## BMP란?

BMP(Bitmap)는 Microsoft에서 개발한 비트맵(Bitmap) 이미지 파일 포멧이다.

이미지의 픽셀 정보를 그대로 저장하는 방식으로 압축을 사용하지 않거나 무손실 압축을 사용하여 원본 화질을 유지한다.

확장자는 `.bmp`를 사용한다.

---

## BMP의 특징

- 비트맵 방식의 이미지 파일이다.
- 원본 화질을 그대로 유지한다.
- 일반적으로 압축을 사용하지 않는다.
- 파일 크기가 큰 편이다.
- Windows 환경에서 널리 사용된다.

---

## BMP의 장점

- 원본 화질을 그대로 유지한다.
- 이미지 품질이 저하되지 않는다.
- 파일 구조가 단순하며 처리하기 쉽다.
- 그래픽 편집에 적합하다.

---

## BMP의 단점

- 파일 크기가 매우 크다.
- 인터넷에서 사용하기에는 비효율적이다.
- 저장 공간을 많이 차지한다.
- 다른 이미지 포멧보다 전송 속도가 느릴 수 있다.

---

## BMP의 활용 분야

- Windows 운영체제
- 이미지 편집 프로그램
- 그래픽 디자인
- 원본 이미지 저장

---

## BMP와 JPG 비교

| 항목 | BMP | JPG |
|------|-----|-----|
| 압축 방식 | 일반적으로 압축 없음 | 손실 없음 |
| 파일 크기 | 매우 큼 | 작음 |
| 화질 | 원본 유지 | 압축 시 화질 손실 |
| 사용 용도 | 편집 및 원본 저장 | 사진 및 웹 이미지 |
| 인터넷 사용 | 적합하지 않음 | 적합 |

---

## BMP와 PNG 비교

| 항목 | BMP | PNG |
|-----|-----|-----|
| 압축 방식 | 일반적으로 압축 없음 | 무손실 압축 |
| 파일 크기 | 매우 큼 | BMP보다 작음 |
| 화질 | 원본 유지 | 원본 유지 |
| 투명 배경 | 지원하지 않음 | 지원 |
| 사용 용도 | 원본 이미지 | 웹 및 UI 이미지 |

---

## BMP 사용 예시

- Windows 기본 이미지 파일
- 그래픽 편집 작업
- 원본 이미지 보관
- 이미지 분석
- 디지털 드로잉

---

## 결론

BMP는 원본 화질을 유지하는 비트맵 이미지 파일 포맷이다.

압축을 거의 사용하지 않아 화질 손실이 없지만 파일 크기가 크며, 주로 이미지 편집이나 원본 보관 용도로 사용된다.
