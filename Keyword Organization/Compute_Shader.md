# Compute Shader

## "Compute Shader"이란?

Compute Shader는 GPU를 이용하여 그래픽 처리뿐만 아니라 일반적인 연산 작업을 수 있도록 만든 셰이더 프로그램입니다.

그래픽 렌더링 과정과 별개로 대량의 데이터를 병렬로 처리할 수 있습니다.

---

## Compute Shader의 특징

- GPU 병렬 연산 사용
- 대규모 데이터 처리 가능
- 그래픽 렌더링과 독립적으로 실행
- 높은 연산 성능 제공
- 실시간 계산에 활용

---

## Compute Shader의 동작 방식

CPU가 작업을 GPU에 전달하면 GPU 내부의 코어가 동시에 데이터를 처리한다.

이를 통해 CPU보다 훨씬 많은 데이터를 빠르게 계산할 수 있다.

---

## Compute Shader의 주요 구성 요소

## Kernel

Compute Shader 내부에서 실제 연산을 수행하는 함수 입니다.

```hlsl
#pragma kernel CSMain
```

---

### Thread Group

스레드들을 묶어서 실행하는 단위입니다.

예 :

```hlsl
[numthreads(8, 8, 1)]
```

## Compute Shader의 활용 분야

### 게임 개발

- 파티클 시스템
- 물리 연산
- 유체 시뮬레이션
- 군중 시뮬레이션

### 그래픽 처리

- 후처리 효과
- 이미지 필터링
- 실시간 조명 계산

### 인공지능

- 딥러닝 연산
- 데이터 분석
- 머신러닝 계산

### 과학 및 공학

- 시뮬레이션
- 수치 해석
- 대규모 연산

---

## CPU와 Compute Shader 비교

| 항목 | CPU | Compute Shader |
|------|------|---------------|
| 처리 방식 | 순차 처리 | 병렬 처리 |
| 코어 수 | 적음 | 많음 |
| 연산 속도 | 상대적으로 느림 | 매우 빠름 |
| 주요 용도 | 일반 연산 | 대량 데이터 계산 |
| 실행 위치 | CPU | GPU |

---

## Compute Shader의 장점

- 매우 빠른 병렬 연산
- GPU 자원을 효율적으로 사용
- 실시간 계산 가능
- 대량 데이터 처리에 유리

---

## Compute Shader의 단점

- 프로그래밍 난이도가 높다.
- GPU 메모리 관리가 필요하다.
- 디버깅이 어렵다.
- 모든 GPU가 동일한 성능을 제공하지 않는다.

---

## Unity에서의 활용

Unity에서는 `.compute` 파일을 사용하며 Compute Shader를 작성할 수 있다.

주요 활용 예시 : 

- GPU 파티클
- 물 시뮬레이션
- 군중 시스템
- 실시간 이펙트
- AI 연산

---

## CUDA와 Compute Shader 비교

| 항목 | CUDA | Compute Shader |
|------|------|---------------|
| 개발사 | NVIDIA | Microsoft DirectX |
| 지원 GPU | NVIDIA 전용 | 다양한 GPU |
| 용도 | 범용 GPU 연산 | 게임 및 그래픽 연산 |
| 사용 분야 | AI, 과학 계산 | 게임, 시뮬레이션 |

---

## 결론

Compute Shader는 GPU의 병렬 처리 능력을 활용하여 대량의 데이터를 빠르게 계산하는 기술입니다.

게임 개발, 그래픽 처리, 인공지능, 물리 시뮬레이션 등 다양한 분야에서 사용되며, 현대 GPU 프로그래밍의 중요한 기술 중 하나입니다.
