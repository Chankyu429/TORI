# Process

## Process란?

Process(프로세스)는 실행 중인 프로그램을 의미한다.

프로그램이 실행되면 운영체제는 메모리를 할당하고, CPU를 사용하여 프로그램을 실행하는데 이 실행 중인 상태를 프로세스라고 한다.

---

## Process의 특징

- 실행 중인 프로그램이다.
- 운영체제로부터 독립된 메모리 공간을 할당받는다.
- 하나 이상의 Thread를 포함할 수 한다.
- 기본적으로 다른 프로세스와 격리되어 독립적으로 실행되지만, 필요 시 IPC(주소 공간 공유, 메시지 큐 등)를 통해 통신할 수 있다.

---

## Process의 구성

- Code 영역
- Data 영역
- Heap 영역
- Stack 영역

---

## Process의 예시

- Chrome 실행
- Visual Studio 실행
- Unity 실행
- 게임 실행

---

## 결론

Process는 운영체제에서 실행 중인 프로그램을 의미하며, 프로그램 실행에 필요한 메모리와 자원을 할당받아 작업을 수행한다.
