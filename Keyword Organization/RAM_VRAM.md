# RAM vs VRAM
## "RAM"이란?
RAM(Random Acccess Menory)은 컴퓨터의 주기억장치로 CPU가 현재 실행중인 프로그램과 데이터를 임시로 저장하는 공간입니다.

컴퓨터를 끄면 저장돈 데이터가 사라지는 휘발성 메모리입니다.

---

## "VRAM"이란?

VRAM(Video Random Access Memory)은 그래픽카드에 탑재된 메모리로 그래픽 처리에 필요한 데이를 저장하는 역할을 합니다.

GPU가 텍스처, 해상도, 영상 데이터 등을 처리할 때 사용됩니다.

---

## RAM의 특징

- CPU가 사용합니다.
- 프로그램 실행에 사용됩니다.
- 멀티테스킹 성능에 영향을 줍니다.
- 용량이 부족하면 컴퓨터가 느려질 수 있습니다.

---

## VRAM의 특징

- GPU가 사용합니다.
- 그래픽 데이터를 저장합니다.
- 게임 성능에 영향을 줍니다.
- 고해상도 텍스처 처리에 필요합니다.

---

## RAM의 활용 분야

- 운영체제 실행
- 웹 브라우저
- 문서 작업
- 게임 데이터 처리
- 프로그램 실행

---

## VRAM의 활용 분야

- 게임 그래픽 처리
- 3D 모델링
- 영상 편집
- 인공지능 연산
- 고해상도 텍스처 처리

---

## RAM과 VRAM 비교

| 항목 | RAM | VRAM |
|------|------|------|
| 사용 장치 | CPU | GPU |
| 역할 | 프로그램 실행 | 그래픽 처리 |
| 저장 데이터 | 프로그램 및 운영체제 | 텍스처, 영상 데이터 |
| 성능 영향 | 멀티테스킹 | 게임 및 그래픽 |
| 위치 | 메인보드 | 그래픽카드 |

---

## RAM 부족 시 발생하는 현상

- 프로그램 실행 속도 저하
- 멀티태스킹 성능 저하
- 프로그램 강제 종료
- 시스템 느려짐

---

## VRAM 부족 시 발생하는 현상

- 프레임 저하
- 텍스처 로딩 지연
- 그래픽 품질 저하
- 게임 끊김 현상

---

## 게임에서의 예시

### RAM

- 게임 데이터 로드
- 백그라운드 프로그램 실행
- 운영체제 관리

### VRAM

- 텍스처 저장
- 그림자 처리
- 해상도 렌더링
- 그래픽 효과 처리

---

## 결론

RAM은 CPU가 사용하는 주기억장치이며, VRAM은 GPU가 사용하는 그래픽 메모리이다.

RAM은 프로그램 실행과 멀티태스킹에 영향을 주고, VRAM은 게임과 그래픽 성능에 영향을 준다.
