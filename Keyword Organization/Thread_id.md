# TCB - Thread ID (TID)

## Thread ID란?

Thread ID(TID)는 TCB(Thread Control Block)에 저장되는 정보 중 하나로, 각 스레드를 구별하기 위해 운영체제가 부여하는 고유한 번호이다.

---

![Thread_id 이미지](../imags/Thread_id.jpg)

---

## Thread ID의 특징

- 스레드마다 고유한 번호를 가진다.
- 운영체제가 자동으로 할당한다.
- TCB에 저장된다.
- 스레드를 식별하는 데 사용된다.

---

## Thread ID의 역할

- 스레드 식별
- 스레드 관리
- 스레드 제어
- 스레드 검색

---

## 활용 예시

- 멀티스레드 프로그램
- 작업 스케줄링
- 디버깅
- 운영체제의 스레드 관리

---

## 결론

Thread ID(TID)는 TCB에 저장되는 고유한 식별 번호로, 운영체제가 각 스레드를 구분하고 관리하는 데 사용된다.
