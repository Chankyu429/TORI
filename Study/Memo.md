# SB_텍스트 기반 게임 구현하기

## 과제에서 사용하게 될 기능들
	- 변수, 배열, 함수 : 데이터 저장과 처리
	- 조건문과 반목문 : 게임 메뉴, 입력, 유효성 검사, 전투 루프
	- 클래스 상속, 다형성 : 캐릭터 및 구조 등의 객체지향 설계
	- List, Dictlomary : 아이템 목록, 인벤토리 등의 제네릭 컨테이너 관리
## 1단계 구현
  - 캐릭터 생성 화면
    ```
    ======================================================================
    플레이어 데이터 입력
    ======================================================================
    플레이어 이름 : Chankyu
    HP, MP : 100, 50
    공격력, 방어력 : 20, 4

    //입력이 완료 되었을때 보일 화면
    =======================================================================
    Chankyu의 상태창
    =======================================================================
    HP : 100, MP : 50, 공격력 : 20, 방어력 : 4
    ========================================================================
	  ```

## 구현해야 할 항목
- Player Class
    - string name
    - float hp, mp
    - float attack, defence
- String 타입으로 이름 입력 받기
- string 타입으로 HP, MP 입력 받기
    - ","로 문자열 배열로 분할해서 HP, MP 저장하기
- string 타입으로 attack, defence 입력 받기
    - ","로 문자열 배열로 분할해서 attack, defence 저장하기
- 저장 결과 보여주기 -[Chankyu의 상태창]

## 2단계
### 입력 검증 화면
```
    ======================================================================
    플레이어 데이터 입력
    ======================================================================
    플레이어 이름 : Cha
    플레이어 이름 : 너무 짧아요, 다시 입력해 주세요.
    플레이어 이름 : Chankyu
    HP, MP : 50, 30
    최소 HP는 60, 최소 MP는 40입니다. 다시입력해 주세요
    HP, MP : 100, 50
    공격력, 방어력 : 10, 3
    최소 공격력은 15, 최소 방어력은 5입니다. 다시입력해 주세요
    공격력, 방어력 : 20, 5
    

    ====================================================================
    Chankyu의 상태창
    ====================================================================
    HP : 100, MP : 50, 공격력 : 20, 방어력 : 5
    
```

- 1단계에서 입력받을 때 값 검증
  - name이 Length가 3이하 일때 다시 입력
  - hp < 50, mp < 40이면 다시입력
  - attack < 15, defence < 5이면 다시 입력
  - 입력 블럭을 while(true)로 감싸기
  - 조건으 ㄹ만족하면 break로 입력 루프 탈출
  - [Chankyu의 상태창] 보여주기

## 3단계
- 상태 관리 메뉴

```
    * HP 포션 3개, MP 포션 3개를 지급합니다.
    * 공격력 Up 쿠폰 2개, 방어력 Up 쿠폰 3개를 지급했습니다.
    ======================================================================
    < Chankyu 강화 >
    1. HP Up
    2. MP Up
    3. 공격력 2배
    4. 방어력 2배
    5. 능력치 보기
    6. 게임 시작 
    ======================================================================
    메뉴를 선택하시오 : 1
    ** HP가 20 증가했습니다.(HP 포션 -1 => 남은 포션 2개)
    메뉴를 선택 하세요 : 3
    ** 공격력이 2배 증가했습니다. (공격력 Up 코폰 -1 => 남은 쿠폰 2개)
    메뉴를 선택하세요.
    ====================================================================
    Chankyu의 상태창
    ====================================================================
    HP : 100, MP : 50, 공격력 : 20, 방어력 : 5
    ====================================================================
    메뉴를 선택 하세요 : 0

    ====================================================================
    Game Start ! ! !
    ====================================================================
```
- HP 포션, MP 포션 변수에 저장
- 공격력 Up 쿠폰, 방어력 Up 쿠폰 변수에 저장
- bool bGameStart = false => 게임이 시작되었는지 판단할 변수
- bool bGameStart 와 select 항목으로 메뉴 루프 구현하기
- HP 포션, MP 포션, 공격력 Up 쿠폰, 방어력 Up 쿠폰 각각 0이면 사용불가
    - 포션이 부족합니다.
    - 쿠폰이 부족합니다.
- 0번이 입력되면 게임 시작
    - bGameStart = true
